# Portfolio — Stephanie Poaty

Site portfolio personnel de Stephanie Poaty, Data Engineer / Data Analyst.

## Stack

Site statique en HTML / CSS / JS, sans dépendances ni build.

## Développement local

Ouvre simplement `index.html` dans un navigateur, ou lance un petit serveur local :

```bash
python -m http.server 8000
```

Puis va sur http://localhost:8000

## Déploiement

Déployé automatiquement sur GitHub Pages via GitHub Actions à chaque push sur `main`.
